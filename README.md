# 🌴 Palm Leaf Disease Detection using Deep Learning

## 📌 Project Overview

This project is a **Deep Learning-based web application** that detects diseases in palm leaves using image classification.
It leverages **Convolutional Neural Networks (CNNs)** and **Transfer Learning** techniques to classify leaf images into healthy or diseased categories.

The application is built using **Streamlit**, providing an intuitive interface where users can upload images and receive instant predictions.

---

## 🚀 Features

* 📷 Upload palm leaf images for analysis
* 🤖 Disease prediction using trained deep learning models
* ⚡ Fast and real-time inference
* 🌐 Clean and interactive Streamlit UI
* 🧠 Supports multiple models for better accuracy

---

## 🧠 Models Used

* Custom CNN Model
* MobileNetV2 (Transfer Learning)

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/palm-leaf-disease-detection.git
cd palm-leaf-disease-detection
```

### 3️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 4️⃣ Run the application

```
python -m streamlit run app.py
```

---

## ☁️ Deployment (Streamlit Cloud)

1. Push your code to GitHub
2. Connect your repository to Streamlit Cloud
3. Configure `requirements.txt` and `runtime.txt`
4. Deploy your application 🚀

## 📦 Model Usage

⚠️ Model files are not included in the repository due to size limitations.

To run this project:

* Train the model using the provided notebook (`SImple_CNN.ipynb`)
  **OR**
* Use your own trained `.h5` model files and place them in the project directory
